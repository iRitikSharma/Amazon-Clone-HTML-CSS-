# Amazon Clone (HTML & CSS)

A simple Amazon homepage clone built using pure HTML and CSS. This project focuses on front-end layout design and responsiveness without using JavaScript.

## Features
- Amazon-style navigation bar
- Search bar and login section
- Product categories and cards
- Multi-column footer like Amazon
- Fully responsive layout

## Technologies Used
- HTML5
- CSS3 (Flexbox & CSS Grid)

## How to Run
1. Download or clone the project.
2. Open `index.html` in your browser.

## Folder Structure
amazon-clone/
│
├── index.html
└── style.css
